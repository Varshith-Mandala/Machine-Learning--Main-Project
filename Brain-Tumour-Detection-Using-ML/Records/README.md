🗄️ Records – Database Storage

This folder contains the database used to store patient information and model prediction results for the Brain Tumor Detection system.

📌 Purpose

The database stores:

Patient Name

Age

MRI Image Path

Prediction Result (Tumor / No Tumor)

Confidence Score

Timestamp

This ensures that every prediction made through the web application is recorded and can be retrieved later.

🛠️ Database Type

SQLite database file:

tumor_records.db
⚠️ Important Note

The actual .db file is not included in the repository for security and size reasons.
