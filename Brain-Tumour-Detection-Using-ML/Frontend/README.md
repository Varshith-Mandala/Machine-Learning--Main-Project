🌐 Frontend – Brain Tumor Detection Web Interface

This folder contains the frontend implementation of the Brain Tumor Detection system.
It provides a user-friendly interface for uploading MRI images and viewing prediction results.

🎯 Purpose

The frontend enables:

MRI image upload

Tumor prediction display

Confidence score visualization

Patient data entry

Interaction with backend API

This layer connects users to the trained deep learning model through a simple web interface.

🛠️ Technologies Used

HTML5

CSS3

JavaScript
